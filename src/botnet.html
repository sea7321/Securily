What is a Botnet Attack?

Malicious software intended to break into and/or damage a computer system (compromise a system) without the owner’s approval


Types of Botnet Attacks

Adware

One of the least risky and most "money spinning" types of malware. Accesses the user's system to automatically display/play an advertisement.

Spyware

Software used to track users internet activities in order to send advertising like adware back to the system. Involves the sale of personal data to a third party.

Virus

Software that attaches itself to another program, which reproduces itself when the program is run. Consumes resources, which slows down the computer. Spread by sharing software/files between computers.

Worm

“Silently” replicates itself in active memory and destroys all data and files on a computer. “Eats” the system operating files and other file data.

Trojan Horse

One of the most dangerous types of malware that have an “advertised” use with a hidden ability. Discovers crucial information or accessibility to the computer system. The user must download and install software in order to get a trojan horse.

Backdoor

A “backdoor” into a computer, which quietly provides a network connection for other hackers or malware for viruses or spam. Hides certain access from the user.

Rootkit

The least detectable type of malicious software, making it the hardest to remove. Serves for evidence removal which can sanitize logs, remove timestamps, install backdoors, mask files and processes… etc.

Keylogging

A spy software that captures all keystrokes on a computer. Records passwords, credit card information, and other sensitive information. Usually transmitted to a file/server for reading use later.

Ransomware

Restricts the user’s access until a ransom is paid. Encrypts the victims files and demands the sum of money to be paid to decrypt the files. Typically use a trojan to enter a system.

Botnet: A collection of internet connected programs communicating with each other to perform a task. All infected systems are typically under control by a single system Tasks include: spam and malware delivery
